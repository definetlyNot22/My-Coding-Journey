## 🚀 Getting Started

### DSA (C++)
```bash
cd dsa
g++ <filename>.cpp -o output
./output
```

### C
```bash
cd c
gcc <filename>.c -o output
./output
```

### HTML Website
Just open the `.html` file directly in your browser, or use a live server extension in VS Code.

### React App
```bash
cd react-app
npm install
npm run dev
```
