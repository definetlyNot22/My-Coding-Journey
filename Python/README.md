# Snake, Water, Gun

A console-based Python game — a twist on the classic Rock-Paper-Scissors, where you play against the computer using Snake, Water, and Gun as your three choices.

## 🛠️ Built With
- **Python 3**
- Built-in `random` module for the computer's choice

## ✨ Features
- Simple text-based input — type your choice (`snake`, `water`, or `gun`)
- Computer randomly selects its own choice each round
- Win/lose/draw logic based on Snake, Water, Gun rules:
  - Snake beats Water
  - Water beats Gun
  - Gun beats Snake
- Displays both players' choices and the round result in the terminal

## 📂 Project Structure
```
Snake-Water-Gun/
│
└── project.py    # Game logic and console interaction
```

## 🚀 Getting Started

### Prerequisites
- Python 3.x

### Installation & Running

1. Clone the repository
   ```bash
   git clone https://github.com/definetlyNot22/Snake-Water-Gun.git
   ```
2. Navigate into the project folder
   ```bash
   cd Snake-Water-Gun
   ```
3. Run the game
   ```bash
   python project.py
   ```

## 🎮 How to Play
- When prompted, type your choice: `snake`, `water`, or `gun`
- The computer picks its move randomly
- The result (win, lose, or draw) is printed to the terminal
